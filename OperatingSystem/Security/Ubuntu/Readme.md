:atom_symbol:**脚本使用:**

描述: 通常针对于才安装的服务器系统，针对ubuntu初始化可以在 `Ubuntu22.04.conf` 中进行相应配置，然后在执行该脚本。

```bash
# 执行权限赋予
chmod +x -R *

# 开始初始化加固
Ubuntu22.04-InitializeReinforce.sh  --start
```
